HTML :

HTML es el lenguaje que se emplea para el desarrollo de páginas de internet. Está compuesto por una seríe de etiquetas que el navegador interpreta y da forma en la pantalla.

HTML dispone de etiquetas para imágenes, hipervínculos que nos permiten dirigirnos a otras páginas, saltos de línea, listas, tablas, etc.

Etiquetas Html

Parrafo = < p > < / p >

Imagenes = < img > < / img >

Link = < a > < / a >

Titulo

  • < h1 > < / h1 >
  • < h2 > < / h2 >
  • < h3 > < / h3 >
  • < h4 > < / h4 >
  • < h5 > < / h5 >
  • < h6 > < / h6 >
