Etiqueta Auido

Una de las principales cosas que llama la atención cuando hablamos de HTML5, son las famosas etiquetas

< audio width="300" height="32" src="micancion.mp3" controls="controls" autoplay="autoplay" preload="">

Atributos de la etiqueta AUDIO

Estos serían los atributos disponibles para la etiqueta AUDIO:

  • height: Para definir la altura del iframe
